# Motivation
원자가 주기적으로 배열되있는 원자표면 atom coordinate를 새로 잡을 일이 많다.
atom coordinate를 잡는 것은 단순 선형 대수학이다.
하지만, 배번 행렬을 잡는 건 여간 귀찮은 일이 아니다.
자잘한 함수 조각들을 많이 만들어봤지만 입출력이 통일되어있지 않아 
매번 재사용이 용이하지 않았다.
통합된 형태의 객체를 만들어 이런 문제를 해결하고자 한다.
